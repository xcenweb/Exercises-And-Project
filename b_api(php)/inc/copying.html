云轩API使用条款
云轩API（以下简称“系统”）是由小岑应用自研并开发的一款API系统。使用本站API即代表你已同意以下条款。

1.在你调用API时，为了防止出现机器人影响其他用户的使用，系统有权记录并存储你的IP地址及相关信息。
2.系统如果发现危险、异常的调用方式，系统有权封禁调用者IP。
（二）异常处理
1.系统不能保证所有接口都不会出现问题，若接口出现异常响应，可能是你已被禁IP或接口出现问题，请发邮件到xcenweb@qq.com反馈问题或在系统内用IP查询系统查询自己是否已被封禁。
2.若你被意外封禁IP，请联系QQ：1610943675解除封禁。
（三）系统维护
1.为保持系统的稳定性，系统会进行定期维护，维护时间一般不会超过3天。
2.系统维护前3天会将公告放置网站主页面或在官方QQ群聊里通知。
3.系统维护时所有API接口均启用备用路径，备用路径速度可能会变慢
4.系统维护时已对接的不用更换API接口！
